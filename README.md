# Conan Exiles Character Planner by The Archivist

A browser-based character planner for **Conan Exiles**, created to help players build, test, compare, and share character setups before committing to them in-game.

This planner includes attribute planning, perk tracking, feat management, gear selection, buffs, armor set testing, weapon estimates, stat totals, randomized lore-friendly character options, and build export tools. It is designed as a practical planning tool for players, roleplayers, server admins, and community builders who want a cleaner way to organize Conan Exiles character builds.

## Features

### Character Planning

Create and customize a character profile with:

* Character name
* Race
* Religion
* Clan name or second name
* Custom build name
* Build type
* Character portrait
* Lore-friendly randomized names and clan names

The planner includes a **Randomize All** option for quickly creating character concepts.

### Attributes and Perks

Plan your attribute spread and see how it affects your final build.

Includes:

* Strength
* Agility
* Vitality
* Authority
* Grit
* Expertise

The planner also supports Conan Exiles perk choice logic. At certain milestone levels, where the game requires choosing between one perk or another, the planner tracks those choices instead of automatically unlocking both.

### Gear and Buffs

Test different armor pieces, weapons, shields, off-hand items, and buffs.

Includes:

* One-handed weapons
* Two-handed weapons
* Daggers, knives, claws, and chakrams
* Ranged weapons
* Shields and torches
* Head, chest, hands, legs, and feet armor slots
* Active buffs and modifiers
* Gear totals
* Armor, weight, durability, damage, penetration, and status effect summaries

Item IDs are displayed to make it easier to identify and correct unclear or mismatched item names.

### Armor Set Matching

The planner attempts to group armor pieces into recognizable full armor sets using item name patterns from the included data.

This helps create cleaner armor set dropdowns while still keeping individual item IDs visible for manual verification.

### Stat Calculations

The planner displays calculated totals for:

* Health
* Stamina
* Carry capacity
* Total armor
* Strength weapon damage
* Agility weapon damage
* Follower damage
* Concussive damage

It also includes a detailed breakdown showing how attributes, gear, and buffs contribute to the final numbers.

### Feat Planner

Track feats and knowledge unlocks with:

* Available feat points
* Spent feat points
* Remaining feat points
* Level requirements
* Prerequisite chains
* Selected feats
* Feat details

The planner can help identify when a feat requires another feat before it can be unlocked.

### Extended Level Support

The tool includes optional support for extended levels beyond the default Conan Exiles level cap.

This is useful for servers that use higher level caps or custom progression settings.

### Save, Load, and Export

Builds can be saved and shared using export tools.

Includes:

* Export build as JSON
* Import build from JSON
* Copy build text
* Export Word document

The Word export is useful for sharing character builds, roleplay concepts, or admin-reviewed character plans.

## Intended Use

This planner is intended for:

* Conan Exiles players planning builds
* Roleplay server players creating character sheets
* Server admins reviewing character setups
* Modded servers using extended levels
* Players testing gear, buffs, and attribute combinations
* Community members sharing builds with others

## Notes

This is an unofficial fan-made tool for Conan Exiles.

The planner uses exported game data and local calculations where available. Some values may vary depending on game updates, server settings, mods, balance changes, or custom server rules.

Item IDs are included so names and values can be checked and corrected more easily during development.
